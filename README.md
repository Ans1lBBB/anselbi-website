# Ansel Bi 個人官網

統一個人 IP 官網，三語獨立頁面 + SEO + Cloudflare 自動部署。

- 繁中：https://www.anselbi.com/zh-tw/
- 簡中：https://www.anselbi.com/zh-cn/
- 英文：https://www.anselbi.com/en/

根路徑 `https://www.anselbi.com/` 依瀏覽器語言自動導向。

## 開發

```bash
npm run build   # 從 content/ 產生靜態頁
```

文案編輯：`content/home.js`、`content/privacy.js`

詳見 [SETUP.md](./SETUP.md)、[APP-STORE-URLS.md](./APP-STORE-URLS.md)
